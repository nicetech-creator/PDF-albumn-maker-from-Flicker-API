Flickr2PDF

Usuage Instructions:
1. Install the necessary libraries:
    pip install flickr_api
    pip install bs4
    pip install weasyprint
    pip install jinja2

2 Run the script by entering the url
    > python flickr2pdf.py https://www.flickr.com/photos/125925288@N02/albums/72157692656165215

3. Wait for it to download the photos and create the PDF
